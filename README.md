# f2all

*A Bulk file renaming script.*

## Usage

```
Usage: f2all [-r] [-d directory] [-b base-name] [-s start-num]
             [-z pad-num] [-g pattern]

  -r                    perform the renaming
  -d directory          directory of files to rename
  -b base-name          base name for all files to use
  -s start-num          starting file number
  -z pad-num            number of zeroes to pad file number with
  -g pattern            glob pattern for file matching
```
